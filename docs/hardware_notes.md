# 硬件接口备注

以下引脚信息来自当前源码，仅供快速阅读工程时参考；实际接线请以你的小车底盘和驱动板原理图为准。

## 电机与舵机

- `PA0` / `PA1`：左右电机 PWM 输出（TIM2）
- `PA2` / `PA3`：左电机方向控制
- `PA4` / `PA5`：右电机方向控制
- `PB0`：舵机 PWM 输出（TIM3 CH3）

## 超声波模块 HC-SR04

- `PA6`：Trig
- `PA7`：Echo

## 串口

- `USART1`
  - `PA9`：TX
  - `PA10`：RX
  - 用途：接收视觉/上位机识别结果

- `USART3`
  - `PB10`：TX
  - `PB11`：RX
  - 用途：连接 JY61P 姿态模块

## 其他

- `PB1`：在 `System/USART3.c` 中被初始化为普通输出，高电平置位；源码注释中把它当作 485 控制脚保留位处理。
